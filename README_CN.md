# Game Macro - AHK游戏宏工具 v0.2.0

一个功能强大的AutoHotkey v2游戏宏工具，支持像素检测、技能循环、自动化操作等功能。

## 项目概述

Game Macro是一个专业的游戏自动化工具，专为需要精确像素检测和复杂技能循环的游戏设计。它采用模块化架构，支持多线程操作、智能规则引擎和实时调试功能。

## 主要特性

### 🎯 核心功能
- **像素检测引擎**：基于DXGI的高性能屏幕像素采集
- **技能循环系统**：支持复杂的技能轮换和条件判断
- **多线程支持**：可配置多个独立执行线程
- **智能规则引擎**：基于条件的自动化决策系统

### 🛠️ 技术特性
- **模块化架构**：清晰的代码组织和易于扩展
- **实时调试**：内置调试工具和日志系统
- **多语言支持**：支持中英文界面切换
- **配置文件管理**：完整的配置导入导出功能

### 📊 高级功能
- **BUFF计时器**：自动监控和管理增益效果
- **技能调试器**：实时查看施法条和技能状态
- **采集诊断**：详细的像素采集性能分析
- **黑屏保护**：智能检测和避免黑屏状态

## 快速开始

### 系统要求
- Windows 10/11 操作系统
- AutoHotkey v2.0 或更高版本
- 管理员权限（用于屏幕采集）

### 安装步骤

1. **下载项目**
   ```bash
   git clone https://github.com/bruceli0305/game-macro.git
   cd game-macro
   ```

2. **安装依赖**
   - 确保已安装AutoHotkey v2
   - 项目包含预编译的DXGI库，无需额外安装

3. **运行程序**
   ```bash
   # 双击运行 Main.ahk
   # 或使用命令行
   AutoHotkey64.exe Main.ahk
   ```

### 基本配置

1. **首次运行**：程序会自动创建必要的配置文件和目录
2. **界面语言**：在设置中切换中英文界面
3. **热键配置**：默认使用F9键开始/停止宏

## 使用指南

### 技能配置
1. 打开"技能"页面
2. 添加技能名称、热键和像素检测位置
3. 配置颜色容差和检测参数

### 循环规则
1. 在"循环规则"页面创建规则
2. 设置触发条件和执行动作
3. 配置优先级和冷却时间

### 多线程管理
1. 在"线程配置"页面管理执行线程
2. 为不同线程分配不同的技能和规则
3. 设置线程优先级和调度策略

## 配置文件说明

### 主要配置文件
- `Config/AppConfig.ini` - 应用全局配置
- `Languages/` - 语言文件目录
- `Profiles/` - 用户配置文件目录

### 配置项示例
```ini
[General]
Language=zh-CN
Version=0.1.3

[Logging]
Level=DEBUG
RotateSizeMB=10
RotateKeep=5
```

## 开发指南

### 模块扩展
项目采用模块化设计，可以轻松添加新功能：

1. **添加新引擎**：在`modules/engines/`目录创建新引擎
2. **扩展UI页面**：在`modules/ui/pages/`添加新页面
3. **自定义规则**：通过RuleEngine扩展条件判断逻辑

### API文档
核心模块提供清晰的API接口：

- `Core_Init()` - 初始化核心系统
- `Logger_Info()` - 日志记录
- `Rotation_Start()` - 启动轮换引擎
- `Pixel_GetColor()` - 像素颜色获取

## 故障排除

### 常见问题

**Q: 程序无法以管理员权限运行**
A: 右键点击Main.ahk，选择"以管理员身份运行"

**Q: 像素检测不准确**
A: 检查颜色容差设置，使用采集诊断工具调试

**Q: 热键冲突**
A: 在热键配置页面修改默认热键

### 日志查看
程序生成详细的日志文件，位于`Logs/`目录：
- 使用"日志查看"页面实时监控
- 或直接查看日志文件进行调试
