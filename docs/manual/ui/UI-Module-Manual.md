## 📖 欢迎使用

欢迎使用Game Macro！本手册将帮助您快速掌握UI界面的使用方法，让您轻松配置和使用游戏宏功能。

**什么是Game Macro？**  
Game Macro是一个专业的游戏自动化工具，可以帮助您自动执行游戏中的重复操作，提高游戏效率。

**为什么要学习UI界面？**  
UI界面是您与Game Macro交互的主要方式。通过它，您可以：
- 配置角色参数和热键
- 管理技能和检测设置
- 设置自动化规则
- 导入导出配置文件
- 监控系统运行状态

## 🎯 界面入门

### 程序启动

**启动步骤**:
1. 双击 `Main.ahk` 文件
2. 程序会在系统托盘显示图标
3. 右键托盘图标，选择"显示主窗口"
4. 主界面将在屏幕中央打开

**界面组成**:
- **左侧导航栏**: 包含所有功能菜单
- **右侧内容区**: 显示当前页面的具体内容
- **顶部标题栏**: 显示程序名称和版本

### 导航操作

**左侧导航说明**:
```
📋 概览与配置    ← 基础设置页面
📊 数据与检测    ← 游戏数据配置
🔄 自动化       ← 自动执行设置  
🛠️ 高级功能     ← 高级配置选项
🧰 工具        ← 导入导出等工具
⚙️ 设置        ← 界面和系统设置
```

**基本操作**:
- **展开菜单**: 点击菜单前的 "+" 号
- **切换页面**: 点击菜单项目名称
- **查看内容**: 右侧会显示对应的设置界面

## 🔧 基础配置（概览与配置页面）

这是您配置Game Macro的第一步，也是最重要的页面。

### 1. 角色配置管理

**创建新角色**:
1. 点击"新建"按钮
2. 输入角色名称（如"战士"、"法师"等）
3. 点击"确定"保存

**切换角色**:
- 点击下拉菜单选择不同角色
- 每个角色有独立的配置

**复制角色**:
1. 选择要复制的角色
2. 点击"复制"按钮
3. 输入新角色名称
4. 新角色会继承原角色所有设置

**删除角色**:
1. 选择要删除的角色
2. 点击"删除"按钮
3. 确认删除（注意：删除后无法恢复）

### 2. 热键设置

**设置开始热键**:
1. 点击"开始热键"输入框
2. 按下您想设置的键（如F1、F2等）
3. 确认设置成功

**设置停止热键**:
1. 点击"停止热键"输入框
2. 按下停止键（如F2、ESC等）
3. 确认设置成功

**鼠标键捕获**:
1. 点击"捕获鼠标键"按钮
2. 按下鼠标按键（左键、右键等）
3. 程序会记录您的鼠标操作

**设置建议**:
- 开始键：F1、F2、F3（容易按到）
- 停止键：ESC、F12（紧急停止）
- 鼠标键：根据游戏习惯设置

### 3. 运行参数设置

**轮询间隔**:
- **作用**: 多久检查一次游戏状态（毫秒）
- **推荐值**: 100-200毫秒
- **说明**: 数值越小越灵敏，但消耗更多资源

**全局延迟**:
- **作用**: 按键操作之间的间隔
- **推荐值**: 50-100毫秒
- **说明**: 避免操作过快被游戏检测

**取色避让**:
- **开启**: 启用颜色检测功能
- **关闭**: 只使用按键操作
- **建议**: 开启以获得更精确的控制

**Y偏移**:
- **作用**: 颜色检测的垂直偏移量
- **范围**: -50到+50
- **使用**: 当颜色检测不准确时调整

## 📊 游戏数据配置（数据与检测页面）

这个页面用来配置游戏中具体的技能、点位和默认行为。

### 1. 技能管理

**添加新技能**:
1. 点击"技能管理"子菜单
2. 点击"添加技能"按钮
3. 填写技能信息：
   - 技能名称：输入技能名称
   - 按键设置：选择触发按键
   - 冷却时间：设置技能冷却（秒）
4. 点击"保存"

**编辑技能**:
1. 在技能列表中选择要编辑的技能
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"保存"确认

**删除技能**:
1. 选择要删除的技能
2. 点击"删除"按钮
3. 确认删除操作

### 2. 取色点位

**添加检测点**:
1. 点击"取色点位"子菜单
2. 点击"添加点位"按钮
3. 设置检测参数：
   - X坐标：屏幕横向位置
   - Y坐标：屏幕纵向位置
   - 目标颜色：要检测的颜色值
   - 容差范围：颜色匹配的容许范围

**调整点位位置**:
1. 点击"调整位置"按钮
2. 鼠标移动到目标位置
3. 按空格键确认当前位置
4. 程序会自动记录坐标

**实时预览**:
- 勾选"实时预览"查看检测效果
- 绿色表示检测成功
- 红色表示检测失败

### 3. 默认技能

**设置默认技能**:
1. 选择"默认技能"子菜单
2. 在技能列表中选择要设置为默认的技能
3. 点击"设为默认"按钮
4. 确认设置

**作用说明**:
- 当所有检测条件都不满足时，使用默认技能
- 确保宏始终有技能可以执行
- 建议选择低消耗的技能

## 🔄 自动化设置（自动化页面）

这个页面用来配置自动执行的规则和逻辑。

### 1. 循环规则

**创建循环规则**:
1. 点击"循环规则"子菜单
2. 点击"新建规则"按钮
3. 设置规则条件：
   - 规则名称：给规则起个名字
   - 触发条件：设置执行条件（如检测到某颜色）
   - 执行技能：选择要执行的技能
   - 优先级：数字越小优先级越高

**条件设置**:
- **颜色检测**: 检测特定位置的颜色
- **时间间隔**: 每隔多长时间执行
- **循环次数**: 重复执行多少次

**规则测试**:
1. 配置完规则后点击"测试"按钮
2. 程序会在游戏中模拟执行
3. 查看测试结果是否正确

### 2. BUFF计时器

**添加BUFF监控**:
1. 点击"BUFF计时器"子菜单
2. 点击"添加BUFF"按钮
3. 设置BUFF信息：
   - BUFF名称：BUFF的名称
   - 检测位置：屏幕上的检测坐标
   - 目标颜色：BUFF激活时的颜色
   - 持续时间：BUFF持续的时间（秒）
   - 续时技能：到期后执行的技能

**BUFF状态**:
- **绿色**: BUFF正常激活
- **红色**: BUFF已失效，需要续时
- **闪烁**: 正在检测BUFF状态

**自动续时**:
- 开启后程序会自动监控BUFF状态
- BUFF即将失效时自动使用续时技能
- 确保重要效果持续存在

### 3. 线程配置

**什么是线程**:
线程就像多个"助手"同时工作，每个线程可以执行不同的任务。

**添加线程**:
1. 点击"线程配置"子菜单
2. 点击"添加线程"按钮
3. 设置线程参数：
   - 线程名称：给线程起个名字
   - 优先级：1-10，数字越大优先级越高
   - 工作间隔：线程检查间隔（毫秒）
   - 分配规则：线程负责执行的规则

**线程管理**:
- **暂停线程**: 临时停止某个线程的工作
- **重启线程**: 重新启动暂停的线程
- **删除线程**: 移除不需要的线程

## 🛠️ 高级功能

这里包含一些高级功能，适合有经验的用户使用。

### 1. 轮换配置

**轮换模式**:
轮换模式可以让技能按顺序轮流使用，避免重复。

**设置轮换**:
1. 点击"轮换配置"子菜单
2. 点击"新建轮换"按钮
3. 添加技能序列：
   - 按顺序添加要轮换的技能
   - 设置每个技能的条件
   - 调整技能顺序

**轨道管理**:
- **主轨道**: 默认的技能轮换序列
- **子轨道**: 特殊条件下的轮换序列
- **跳转规则**: 设置何时切换到子轨道

### 2. 采集诊断

**DXGI采集**:
- 用来截取游戏画面进行检测
- 状态显示：正常/异常/未启动
- 如果异常，检查游戏是否全屏运行

**ROI检测**:
- 感兴趣区域检测
- 可以提高检测效率
- 减少不必要的画面分析

**性能统计**:
- 显示检测次数和成功率
- 帮助优化配置参数
- 识别性能瓶颈

## 💡 使用技巧和建议

### 新手建议

**第一次使用**:
1. 从"概览与配置"开始，先设置基本参数
2. 创建一个简单的角色配置
3. 逐步添加技能和检测点
4. 使用测试功能验证设置
5. 最后开启自动化功能

**配置原则**:
- **简单开始**: 不要一开始就设置太复杂
- **逐步调试**: 每次只修改一个参数
- **记录变化**: 记录哪些设置效果更好
- **多测试**: 在不同游戏情况下测试

### 性能优化

**提高效率**:
- 合理设置轮询间隔，不要过于频繁
- 使用取色避让功能提高精确度
- 适当设置延迟避免被检测
- 定期清理不需要的配置

**资源管理**:
- 及时暂停不使用的线程
- 定期检查和清理日志文件
- 避免同时运行过多规则
- 监控系统资源使用情况

### 安全使用

**避免检测**:
- 使用合理的延迟时间
- 不要设置过于频繁的操作
- 模拟人类操作模式
- 定期更换热键

**风险管理**:
- 在测试模式下先验证设置
- 备份重要配置文件
- 了解游戏的相关规定
- 适度使用自动化功能

## ❓ 常见问题解答

### 界面问题

**Q: 程序启动后界面没有显示？**  
A: 检查系统托盘，找到Game Macro图标，右键选择"显示主窗口"

**Q: 界面显示异常或乱码？**  
A: 尝试重启程序，或在设置中切换界面语言

**Q: 左侧导航菜单点击没反应？**  
A: 刷新界面或重启程序，检查配置文件是否损坏

### 配置问题

**Q: 热键设置无效？**  
A: 检查热键是否与其他程序冲突，尝试使用不同的键位

**Q: 取色检测不准确？**  
A: 调整Y偏移值，增大容差范围，或重新设置检测点

**Q: 技能执行顺序混乱？**  
A: 检查规则优先级设置，确保高优先级规则正确配置

### 性能问题

**Q: 程序运行卡顿？**  
A: 增加轮询间隔，暂停不必要的线程，关闭其他占用资源的程序

**Q: CPU使用率过高？**  
A: 降低检测频率，优化检测区域，及时清理日志文件

**Q: 内存占用持续增长？**  
A: 重启程序，检查是否有内存泄漏，定期清理临时文件

### 功能问题

**Q: 自动化功能不执行？**  
A: 检查所有规则是否启用，热键是否正确设置，线程是否正常运行

**Q: BUFF监控失效？**  
A: 验证检测点位置和颜色值，确保游戏窗口保持活动状态

**Q: 导入导出失败？**  
A: 检查文件路径权限，确保有写入权限，验证文件格式正确
